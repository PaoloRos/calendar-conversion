Event model¶

Internal event representation shared by all conversion components.

class calendar_conversion.event.Event(id, summary, all_day=False, start_date=None, start_time=None, end_date=None, end_time=None, location='', description='', timezone=zoneinfo.ZoneInfo(key='Europe/Rome'))¶

calendar_conversion.csv_reader.read_events_csv(source)¶

Read events from a path or an open text stream.

Dates must use ISO YYYY-MM-DD format. Times accept HH:MM or HH:MM:SS. Boolean values are case-insensitive and may be represented by true/false, yes/no, y/n, or 1/0.

Parameters:

source (str | PathLike[str] | TextIO)

Return type:

calendar_conversion.xlsx_reader.read_events_xlsx(source, *, sheet_name=None)¶

Read events from one worksheet in an XLSX path or binary stream.

The active worksheet is read by default. Pass sheet_name to select a worksheet explicitly. Completely empty data rows are ignored.
